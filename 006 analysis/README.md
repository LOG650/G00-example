# 006 analysis

Denne mappen samler analysearbeidet i fase 3 av prosjektet.

## Prinsipp

- Alt teknisk analysearbeid organiseres etter aktivitetene i prosjektplanen.
- Hver aktivitetsmappe har egne `scripts`, `figurer` og `resultat`.
- Milepæler dokumenteres i `milepaeler/` uten å kopiere analysefiler.
- Ett felles `uv`-oppsett i roten brukes av alle aktivitetene.

## Kjøring

Eksempel for aktivitet 3.1:

```powershell
uv run python aktiviteter/3_1_rense_og_strukturere_data/scripts/01_generate_visuals.py
```
